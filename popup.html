🎵 Playlist Manager

Reorder your YouTube playlists

✨ Features

  • ↕️ Reverse playlist order
  • 🔀 Shuffle videos randomly
  • 🔤 Sort alphabetically
  • ⏱️ Sort by duration
  • 📅 Sort by upload date
  • 🔄 Reset to original order

📋 How to Use

1. Go to any YouTube playlist page

2. Look for the floating 🎵 button in the bottom-right corner

3. Click the button to open the options menu

4. Select any sorting option to reorder your playlist

5. Use "Reset" to restore original order
